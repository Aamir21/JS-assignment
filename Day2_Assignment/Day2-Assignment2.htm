Methods related to Array!

There are several methods used for array and variable operations like
  1. concat() : This method combine two or more array into one after another.
  2. copyWithin(): This method copies array elements to another position in the array, overwriting the existing values. This method will never add more items to the array.
  3. push() : This method adds new items to the end of an array, and returns the new length.
  4. pop() : The pop() method removes the last element of an array, and returns that element.
  5. findIndex() :This method returns the index of the first element in an array that pass a test (provided as a function).
